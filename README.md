# 🤖 Bedolaga Auto-Update Script

Автоматический скрипт для бэкапа и обновления системы Bedolaga (Telegram-бот + Кабинет + Caddy).

## ✨ Возможности

- 🔒 **Полный бэкап** базы данных, Redis, конфигов и исходного кода
- 🔄 **Автоматическое обновление** бота, кабинета и Caddy
- ✅ **Проверка здоровья** всех контейнеров после обновления
- 📊 **Цветной отчёт** с итогами работы
- 📝 **Логирование** всех действий в файл
- 🔐 **Безопасный ввод** пароля (не хранится в файле)
- ⚙️ **Интерактивная настройка** при первом запуске
- 🗑️ **Авто-ротация** старых бэкапов
- 📦 **Проверка файлов** бэкапа на целостность

# Запусти скрипт (из любой папки!)
